- understanding units
    - px : not changes
    - % : changes
    - vw, vh : viewport width, viewport height (differ acc to the screen) | changes
    - vmax, vmin : viewport maximum(vw(when h<w) ; to vh(when h>w)), viewport minimum (vh(when h<w) ; to vw(when h>w)) | changes
    - em, rem : em(element font size wrt the parent element), rem(element font size wrt the root element)
